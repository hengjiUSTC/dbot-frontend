## 👀代码使用

### Getting started

1. Clone the repository.

    ```bash
    git clone https://github.com/IsaiaPhiliph/vite-reactts-tailwind-rtk-eslint.git
    ```

2. Install dependencies.

    ```bash
    npm i
    ```

3. Start the development server.

    ```bash
    npm run dev
    ```

4. Build for production.

    ```bash
    npm run build
    ```
